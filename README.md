# C++ CMake Template
A template for C++ project using cmake. 

## Motivation
Make bootstrapping a C++ project easier.

### Prerequisites
* [cmake](https://cmake.org/)

## Build
```
mkdir build && cd -
cmake .. && make -j
```

### Unit tests
Unit tests are built on [Catch2](https://github.com/catchorg/Catch2).

To run tests simply run:
```
make test
```
